## Booking Demo
Create Demo Booking Room in Our Company

#### Structure
- Java 8
- Spring boot 2.5.2
- Postgres 12

#### Before start

I used migrate plugin to create Database. Please use command to run and create Database:
- DB config
  ````
  password: postgres
  username: postgres
  driver: org.postgresql.Driver
  url: jdbc:postgresql://localhost:5432/booking
  ````
- Command line
    ```
    mvn flyway:migrate
    ```
